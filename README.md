# Assignment 2: 2-Player Chess

## How to Run

### Prerequisites

* Windows or UN*X
* JRE 1.8 + (Java 8 or higher)

### Compile in Windows

    gradlew clean
    gradlew jar
    
### Compile in UN*X (Linux/macOS/BSD)

    ./gradlew clean
    ./gradlew jar

### Run

    java -jar build/libs/chess-1.0.jar
